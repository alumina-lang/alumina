Module std::math

Math functions

Note that many standard math functions such as sin and sqrt are defined as methods on the f32 and f64 types.

  • fn div_floor<T>(a: T, b: T) -> (T, T)
    T: Integer

    source

    Calculates the quotient and remainder of integer division

    Unlike the / operator which rounds towards zero, this function always rounds towards negative infinity. This is similar to how // and % work in Python.
