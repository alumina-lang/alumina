Macro std::fmt::format_args

macro format_args($fmt, $args...) { ... }

source

Constructs an array of pieces from a format string and arguments.

This macro is implemented in the compiler, and is used as a foundation for other formatting macros, such as format and println. You probably want to use one of those formatting macros instead. 

format_args!("{} = {}", a, b);
// Expands to:
[
    ::std::fmt::internal::format_arg(&a),
    ::std::fmt::internal::format_arg(&" = "),
    ::std::fmt::internal::format_arg(&b)
]

See also format_arg.

As the type-erased &dyn Formattable<_> wraps a pointer to the original arguments, it is generally not safe to store the array that is returned by format_args.

Example

use std::fmt::{format_args, Formattable, StreamFormatter};
use std::io::StdioStream;

let stdout = StreamFormatter::new(&StdioStream::stdout());

// Type hint is required here in order to infer the formatter type
let args: [&dyn Formattable<StreamFormatter<StdioStream>>; 5]
    = format_args!("{} + {} = {}", 1, 2, 3);

args[0].fmt(&stdout); // "1"
args[1].fmt(&stdout); // " + "
args[2].fmt(&stdout); // "2"
args[3].fmt(&stdout); // " = "
args[4].fmt(&stdout); // "3"
