Struct std::thread::internal::ThreadHandleFunc

struct ThreadHandleFunc<F, T> { ... }

source

An aliased version of ThreadHandle that also contains the function to be called.

If the function is not a closure (or a function pointer), then F is zero-sized and this struct has identical layout to ThreadHandle, otherwise it will contain the captured variables. This aliasing allows us to type-erase the thread function and not have to carry the function type in JoinHandle.
