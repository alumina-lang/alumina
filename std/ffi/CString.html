Struct std::ffi::CString

struct CString { ... }

source

A wrapper for a null-terminated (C) string.

Allocates memory for the string on the heap, so it must be freed.

Example

use std::ffi::CString;
let s = CString::from_slice("Hello, world!");
defer s.free();

libc::puts(s.ptr);
