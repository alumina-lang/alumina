Struct json::JsonValue

struct JsonValue { ... }

source

A JSON value

JsonValue is a discriminated union of all possible JSON values. It can be created using the string, number, boolean, object, array and null methods.

Depending on the JsonValue may hold an allocation, so it should be freed when it is no longer needed.

  • fn object(value: HashMap<StringBuf, JsonValue>) -> JsonValue

    source

    Create a new JSON value from an object

    The object is moved into the JSON value. As a HashMap is used to store the object, the order of the keys is indeterminate.

    Example

    use std::string::StringBuf;
use std::collections::HashMap;
use json::JsonValue;

let obj: HashMap<StringBuf, JsonValue> = HashMap::new();
obj.insert(StringBuf::from_slice("foo"), JsonValue::number(42));
obj.insert(StringBuf::from_slice("bar"), JsonValue::boolean(true));

let value = JsonValue::object(obj);
defer value.free();

println!("{}", value); // {"foo":42,"bar":true} or {"bar":true,"foo":42}

  • fn as_number<T = i32>(self: &JsonValue) -> Option<T>
    T: Numeric

    source

    Retreive a number from this JSON value

    If the JSON value is not a number, Option::none() is returned.

    This method is generic over all numeric types, so it can be used to retrieve JSON numbers as integers, floats, etc. If the number is not exactly representable as the requested type, Option::none() is also returned.

    Example

    use json::builder::into;

let value = 420.into!();

assert_eq!(value.as_number::<i32>(), Option::some(420));
assert_eq!(value.as_number::<f32>(), Option::some(420.0f32));
assert_eq!(value.as_number::<u8>(), Option::none());

let value = true.into!();
assert_eq!(value.as_number::<i32>(), Option::none());
