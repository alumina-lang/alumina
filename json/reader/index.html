    Unescape a JSON string in place.

    This function takes a string that has been escaped according to the JSON specification and unescapes it. The input string must be a valid JSON string, i.e. it must start and end with double quotes. It returns the length of the unescaped string.

    Example

    use json::reader::unescape_string;

let s: [u8; 24] = "\"hello\\uD83D\\uDE00world\"".to_array();
let len = unescape_string(&s).unwrap();
