Struct json::writer::JsonFormatter

source

JSON string formatter

This struct is used to format JSON data into a string. It can be used to write JSON data to a string buffer, a file, or any other destination that implements the Formatter trait.

The formatter can be used in two modes: pretty and compact. In pretty mode, the formatter will add newlines and indentation to make the output more human-readable. In compact mode, the output will be as small as possible.

Examples

use std::string::StringBuf;
use json::writer::JsonFormatter;

let buf = StringBuf::new();
let formatter = JsonFormatter::new(&buf);

formatter.write_object_start().unwrap();
formatter.write_object_key("hello").unwrap();
formatter.write_array_start().unwrap();
formatter.write_null().unwrap();
formatter.write_number(42).unwrap();
formatter.write_string("world").unwrap();
formatter.write_array_end().unwrap();
formatter.write_object_end().unwrap();

assert_eq!(buf[..] as &[u8], "{\"hello\":[null,42,\"world\"]}");
