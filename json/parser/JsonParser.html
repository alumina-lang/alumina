source

JSON parser

This parser takes an iterator of raw tokens from a lexer or another source and converts them into a JsonValue. Meant to be used in tandem with JsonReader.

The parser does not consume the iterator, so it can be used to parse multiple JSON values from the same source (e.g. a stream of JSON values, such as JSON Lines)
