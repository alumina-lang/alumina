source

Extension mixin for JsonWriter protocol

This mixin provides convenience methods for writing JSON values.

Provided methods

  • fn write_number<T>(self: &mut Self, value: T) -> Result<()>
    T: Numeric

    source

    Writes a JSON numeric value

    Default implementation tries to cast it to f64 and calls write_floatmethod. If the number cannot be represented exactly without loss of precision, an error is returned.

    Writers can override this method to provide more efficient implementations.
